# Prompts

## Descrizione
Raccolta di prompt per uso comunitario con modelli AI.  
L’obiettivo è condividere idee utili e migliorabili, permettendo a chiunque di adattarli alle proprie esigenze e contribuire al miglioramento collettivo.

## Indicazioni di uso
- **Uso gratuito**: Tutti i prompt possono essere utilizzati liberamente.
- **Testare sempre i prompt**: Prima di applicarli in progetti o conversazioni importanti, verifica che funzionino come desiderato.
- **Migliorare secondo le proprie esigenze**: Modifica i prompt per renderli più adatti al tuo contesto.
- **Condividere modifiche**: Se apporti miglioramenti utili, contribuisci al repository per arricchire la comunità.

## Struttura dei prompt
I prompt sono organizzati in cartelle per categoria, ad esempio:
